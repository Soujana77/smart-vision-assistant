# Smart Vision Assistant

AI-powered smart vision assistant for visually impaired people using TensorFlow and computer vision for real-time object detection, obstacle awareness, and voice-based navigation assistance.

## Features
- Real-time object detection
- Webcam-based environment monitoring
- Voice alerts for detected objects
- Obstacle awareness system
- TensorFlow-based AI model

## Tech Stack
- Python
- TensorFlow
- OpenCV
- NumPy
